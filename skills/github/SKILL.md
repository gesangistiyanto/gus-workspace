# GitHub Skill

Skill ini dipakai untuk workflow GitHub dasar:
- set remote
- commit perubahan
- push ke `main` / `master`
- sinkron branch

Catatan:
- Gunakan SSH remote (`git@github.com:owner/repo.git`) jika memungkinkan.
- Jangan commit secret mentah.
