# Travily Skill

Skill ini untuk engine Tavily/Travily sebagai alternatif/fallback pencarian.

## Konfigurasi
- urutan engine: `~/.config/search-manager/config.json`
- keys: `~/.openclaw/search_keys.json`

## Contoh urutan
```bash
skills/search-manager/scripts/search_manager set-order travily,serper
```
